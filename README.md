# Open Stack – Das OSH Learning Game

**Open Stack** ist ein interaktives, kartenbasiertes Lernspiel für zwei Spieler, das die Prinzipien und die Dynamik von **Open Source Hardware (OSH)** spielerisch vermittelt. Das Spiel kombiniert Ressourcenmanagement mit strategischer Zusammenarbeit und zeigt auf, wie offene Standards Innovationen beschleunigen können.

---

## Kernkonzept
In einer Welt begrenzter Budgets konkurrieren und kooperieren zwei Akteure bei der Entwicklung technischer Projekte. Das Spiel verdeutlicht den Unterschied zwischen geschlossener Entwicklung (teurer Eigenaufwand) und Open Source (effiziente Nachnutzung und gemeinschaftliche Verbesserung).

## Hauptfunktionen

* **Projekt-Rezepte:** Spieler bauen reale OSH-Projekte (z. B. Smart Home Hubs oder 3D-Drucker) anhand einer Liste benötigter Komponenten nach.
* **Wirtschaftssystem:** Ein integriertes Budget-System simuliert die Kosten von Forschung und Entwicklung sowie die finanzielle Unterstützung durch Förderungen (Grants).
* **Forschungs-Mechanik:** Spieler können gezielt in neue Technologien investieren. Über einen Auswahl-Mechanismus ("Drafting") entscheiden sie, welche Erkenntnisse sie exklusiv nutzen und welche sie der Community im öffentlichen Repository zur Verfügung stellen.
* **Forking & Kooperation:** Das Spiel erlaubt die rechtmäßige Übernahme („Fork“) von Designs anderer Spieler. Zudem können Kooperationen mit wissenschaftlichen Instituten (Uni-Koop) eingegangen werden, um Entwicklungsengpässe gegen ein Entgelt zu überbrücken.
* **Wissensvermittlung:** Ein integrierter Informationsticker liefert während des Spielverlaufs kontinuierlich Fakten zu Lizenzen, Nachhaltigkeit und der gesellschaftlichen Bedeutung von Open Source Hardware.

## Lernziele
Das Spiel vermittelt ohne technische Hürden:
1.  Warum das **Teilen von Bauplänen** ökonomisch und zeitlich vorteilhaft sein kann.
2.  Die Bedeutung von **Standardkomponenten** für die globale Hardware-Entwicklung.
3.  Die Rolle von **Lizenzen und Dokumentationen** für die Langlebigkeit und Reparierbarkeit von Produkten.

## Über das Projekt
Dieses Spiel wurde entwickelt, um die Sichtbarkeit und das Verständnis für offene Hardware-Ökosysteme zu fördern. Es ist im Projekt **OSHOP** entstanden. OSHOP ist ein Teil der Initiative T!Raum und wird durch das Bundesministerium für Forschung, Technologie und Raumfahrt gefördert.
